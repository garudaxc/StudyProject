## 二阶魔方的结构

### 位置编号（标准魔方记号）

面对魔方时，8个角块的命名：

| 编号 | 记号 | 位置描述 |
|------|------|----------|
| 0 | UFR | Up-Front-Right (上右前) |
| 1 | URB | Up-Right-Back (上右后) |
| 2 | UBL | Up-Back-Left (上左后) |
| 3 | ULF | Up-Left-Front (上前左) |
| 4 | DFL | Down-Front-Left (下前左) |
| 5 | DLB | Down-Left-Back (下左后) |
| 6 | DBR | Down-Back-Right (下后右) |
| 7 | DRF | Down-Right-Front (下右前) |

### 角块朝向

每个角块有3个贴纸，可以朝向3个方向（0, 1, 2）：
- **朝向0**：参考贴纸朝上或朝下（标准位置）
- **朝向1**：顺时针扭转120°
- **朝向2**：顺时针扭转240°（或逆时针120°）

> 注意：角块扭转必须是3的倍数才能回到原位，因为角块是三面体。

---

## 核心概念：群作用视角

基本操作（F, B, U, D, L, R）构成群 $G$，魔方状态构成集合 $X$。

每个群元素 $g \in G$ 通过置换作用在状态 $x \in X$ 上，产生新状态 $g \cdot x$。

这是整个理论的**核心**：我们不是"移动魔方"，而是"群元素作用在状态空间上"。

### 步骤2：计算理论状态空间大小

#### 初步估计
- 8个角块的位置：$8!$ 种排列
- 每个角块3种朝向：$3^8$ 种扭转组合
- **无约束总数**：$8! \times 3^8$

请计算这个数值：
```
8! × 3^8 = ?
```

#### 合法状态的约束条件

并非所有状态都是**合法可达**的。存在两个独立约束：

**约束A：角块朝向守恒律**
- 所有8个角块的朝向之和必须 $\equiv 0 \pmod 3$
- 这意味着只有7个角块的朝向是自由的，第8个被确定
- **因子**：$1/3$

**约束B：整体旋转等价**
- 将整个魔方在空间中旋转（24种方式）不产生新状态
- **因子**：$1/24$

#### 计算合法状态数

**公式**：

$$|G_{2\times2}| = \frac{8! \times 3^7}{24} = 3,674,160$$

或者（固定魔方方向）：

$$|X_{total}| = 8! \times 3^7 = 88,179,840$$

请验证这个计算，并解释为什么除以24是可选的（取决于我们是否考虑"带标签的魔方"）。

### 步骤3：设计Python数据结构

用Python定义一个类 `PocketCube`，使其能表示：
- 每个角块的当前位置（置换）
- 每个角块的当前朝向（扭转）

#### 推荐编码方案

```python
class PocketCube:
    """
    二阶魔方状态表示
    
    属性:
        permutation: List[int] - 长度为8的列表，表示每个位置的角块编号
                                   permutation[i] 表示位置i上现在是哪个角块
        orientation: List[int] - 长度为8的列表，表示每个位置角块的朝向
                                   orientation[i] ∈ {0, 1, 2}
    """
    
    def __init__(self, permutation=None, orientation=None):
        # 默认状态：已还原（每个角块在自己的位置上，朝向为0）
        if permutation is None:
            self.permutation = list(range(8))  # [0, 1, 2, 3, 4, 5, 6, 7]
        else:
            self.permutation = permutation
            
        if orientation is None:
            self.orientation = [0] * 8  # [0, 0, 0, 0, 0, 0, 0, 0]
        else:
            self.orientation = orientation
```

### 3. 魔方群阶数推导

二阶魔方群阶数公式推导：

$$|G_{2\times2}| = \frac{8! \times 3^7}{24} = 3,674,160$$

各项含义：
- $8!$：8个角块的位置置换数
- $3^7$：7个角块朝向自由，第8个被约束（朝向总和 mod 3 = 0）
- $24$：魔方整体旋转的等价类（24种空间旋转）
