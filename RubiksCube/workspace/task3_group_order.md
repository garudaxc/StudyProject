# 子任务3：解释角块旋转值定义与群作用对应

## 目标
回答两个关键问题：  
1) 为什么角块朝向要定义为 `0/1/2 (mod 3)`；  
2) 为什么状态参数 `S=(permutation, orientation)` 能与“魔方操作群元素的作用”严格对应。

---

## 1. 你要完成的核心问题

围绕下面四个问题写出解释链路：

1. 角块为什么天然对应三元朝向（`0/1/2`）而不是二元或连续量？
2. 在固定参考系（U/D 参考面）下，`orientation=0` 的判定规则是什么？
3. 单次面转（尤其 `R/L/F/B`）为什么会引起“部分角块 +1、部分角块 -1（即 +2）”的模3变化？
4. 为什么 `S=(p,o)` 与操作序列（群元素）之间能建立稳定、可计算的映射关系？

---

## 2. 建议解释步骤（可直接照此书写）

### 步骤A：先给出“朝向值为何是 mod 3”
- 每个角块是三面体，围绕自身轴旋转 120° 才切换到下一种朝向；
- 三次旋转回到原朝向，因此朝向群是 `Z3`；
- 这就是 `orientation ∈ {0,1,2}` 的几何来源。

### 步骤B：明确参考系与 `orientation=0` 规则
- 采用固定参考系（本项目沿用 U/D 参考）；
- 约定：角块上 U/D 颜色贴纸位于 U/D 面时记为 0；
- 从 0 出发，按固定方向定义 120° 扭转为 +1，再扭转为 +2。

### 步骤C：解释面转如何作用在 `(p,o)` 上
- 置换部分：`new_p[dest] = old_p[src]`；
- 朝向部分：`new_o[dest] = (old_o[src] + delta[dest]) % 3`；
- 解释 `U/D` 常见为 `delta=0`，`R/L/F/B` 对被影响角块产生非零 `delta`；
- 说明“+1 与 +2 成对出现”使总和模3守恒。

### 步骤D：建立“群元素 ↔ 状态变换”对应
- 把每个基本操作看作状态集合上的双射；
- 操作复合对应函数复合，恒等操作对应恒等映射，逆操作对应逆映射；
- 因而任意操作序列（群元素）都可表示为对 `(p,o)` 的确定性变换；
- 说明这就是后续 BFS 在 Cayley 图上搜索的数学基础。
